# Stack

- Stack can be implemented using a dynamic array.
- LIFO - Last In, First Out:
    - The last element that was added in the stack is the first element to be removed.

## Complexity
- 
    | Operation | Big-O Time |
    |---|---|
    | Push | O(1) |
    | Pop | O(1) |
    | Peek/Top | O(1) |
