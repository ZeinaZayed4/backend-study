# CLI Bookmark Manager

#### Description:

- A command-line bookmark manager that fetches page titles automatically, suggests AI-powered tags, supports fuzzy search, tracks visits, and detects dead links.

#### Features:

- Auto title fetching with BeautifulSoup
- AI tag suggestion via Claude API
- Fuzzy search with SequenceMatcher
- Visit tracking and stats
- Dead link detection

#### How to run:

- pip install -r requirements.txt
- python main.py

#### How to test:

- pytest
